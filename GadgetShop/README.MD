# GadgetStore — Магазин портативных гаджетов

Веб-приложение на базе Django для продажи портативных гаджетов: 

## О проекте
Приложение позволяет пользователям просматривать каталог товаров, управлять корзиной и оформлять заказы, а администраторам — гибко управлять ассортиментом через встроенную панель управления.

---

## Список страниц магазина

* **Главная страница (`/`)** 
* **Каталог (`/catalog/`)** 
* **Страница товара (`/product/<slug>/`)** 
* **Корзина (`/cart/`)**
* **Оформление заказа (`/checkout/`)** 
* **Личный кабинет (`/profile/`)** 
* **Страница поиска (`/search/`)**
